# Microsoft Entra ID credentials

You can use these credentials to authenticate when using the HTTP Request node to make a [Custom API call](/integrations/custom-operations/).

## Authentication methods

n8n supports Microsoft Entra ID authentication with OAuth2.

## Related resources

Refer to [Microsoft Entra ID's documentation](https://www.microsoft.com/en-us/security/business/identity-access/azure-active-directory){:target=_blank .external-link} for more information about the service.
